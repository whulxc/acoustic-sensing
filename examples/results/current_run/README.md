# Current Run Outputs

These files were generated by the refactored FFT-correlation pipeline using `data/example/Record.mat`.

- `acoustic_echo_result.png` - exported visualization image.
- `acoustic_echo_result.fig` - MATLAB figure from the same run.
- `time_distance_cc_processed.png` - processed large-scale time-distance CC map.
- `time_distance_cc_unprocessed.png` - unprocessed large-scale time-distance CC map.
- `correlation_map.txt` and `correlation_map.mat` - time-distance correlation map.
- `distance_axis.txt` and `distance_axis.mat` - distance axis for the correlation map.
- `time_axis.txt` and `time_axis.mat` - time axis for the correlation map.
- `CCvalue.txt`, `Disvalue.txt`, `Tvalue.txt` - legacy-compatible text output names.

The default pipeline stops at the large-scale CC-map stage. Tracking and distance-frequency analysis are kept in `matlab/advanced_analysis/` as optional follow-up code.
