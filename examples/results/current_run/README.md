# Current Run Outputs

These files mirror the data products from the MATLAB script's current example run.

- `acoustic_echo_result.png` - exported visualization image.
- `acoustic_echo_result.fig` - MATLAB figure from the same run.
- `correlation_map.mat` - original `CCvalue.mat`.
- `distance_axis.mat` - original `Disvalue.mat`.
- `time_axis.mat` - original `Tvalue.mat`.
- `CCvalue.txt`, `Disvalue.txt`, `Tvalue.txt` - text exports from the original output directory.
- `tracking_result.txt` - exported tracking result.
- `tracking_y_position.txt` - exported tracking y-position result.

The source `CCvalue.txt` file is empty in the original output directory, so the numeric correlation map is preserved in `correlation_map.mat`.
